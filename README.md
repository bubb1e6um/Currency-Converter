#Currency Converter

Веб-приложение для конвертации валют в реальном времени. Проект имеет современный дизайн, адаптированный под мобильные устройства и десктопы.

Особенности

- Актуальные курсы валют: Данные подтягиваются в реальном времени через API.
- Динамические флаги: При выборе валюты флаг страны меняется автоматически.
- Функция Swap: Удобная кнопка для быстрой смены валют местами.
- Адаптивный дизайн: Корректное отображение на телефонах, планшетах и ПК.
- Минимализм: Чистый интерфейс без лишних элементов.

Технологии

Проект написан на чистом (Vanilla) стеке технологий без использования тяжелых фреймворков:

- HTML5 — семантическая структура.
- CSS3 — стилизация (Flexbox, CSS Variables, Media Queries).
- JavaScript (ES6+) — логика работы, асинхронные запросы (Async/Await).

API

В проекте используются следующие бесплатные API:

1.  **[ExchangeRate-API](https://www.exchangerate-api.com/)** — для получения текущих курсов валют.
2.  **[FlagsAPI](https://flagsapi.com/)** — для отображения флагов стран.

Как запустить проект

Этот проект не требует установки зависимостей (npm, yarn) или бэкенда.

1.  **Скачайте** репозиторий:
    ```bash
    git clone [https://github.com/ваш-юзернейм/currency-converter.git](https://github.com/ваш-юзернейм/currency-converter.git](https://github.com/bubb1e6um/Currency-Converter)
    ```
2.  **Откройте** файл `index.html` в любом современном браузере.
